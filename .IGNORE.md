## Factory
### make factory
php artisan make:factory PostFactory
### run tinker
php artisan tinker
### create 50 posts
```
$post = App\Models\Post::factory()->count(50)->create();
```
### create 10 posts with user id
```
$post = App\Models\Post::factory()->count(10)->create([
    'author_id' => \App\Models\User::factory(),
]);
```
